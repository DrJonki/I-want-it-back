Detailed Description

Class that emits particles with customizable initial conditions.

This emitter is universal with respect to the initial conditions of each emitted particle. It works with callbacks that return initial values for the particle attributes (position, rotation, color, ...). So you can pass constants, random distributions, or any functions that compute the value in an arbitrary way.
